### **Name**
TabOrigami

### **Short Description**
Organize cluttered browser tabs intelligently based on context using Gemini AI.

### **Detailed Description**
TabOrigami is a browser extension that uses Gemini AI to transform a cluttered tab bar into a structured, organized workspace. By analyzing the titles, URLs, and content of your open pages, it automatically groups them according to your preferred workflow style.

[ CORE FEATURES ]

- AI-Driven Categorization: Choose from four distinct organization styles—Auto, Task Focus, Work-Life, or Triage—to group tabs based on their content and purpose.
- Interactive Preview: Review AI suggestions before applying them. Use drag-and-drop to easily reorder tabs or move them between groups, and use checkboxes to fine-tune the final selection.
- Smart Cleanup: Automatically identifies and closes redundant "New Tab" pages, while keeping tabs from the same domain together.
- Reliable Undo: Instantly revert to your previous tab state with a single click if you change your mind after organizing.
- Pinned Tab Control: Optionally exclude pinned tabs from the organization process to keep your fixed workflow tabs untouched.
- Real-time AI Progress: View the AI's tab analysis progress, keeping you informed during the categorization process.
- Privacy-Focused (BYOK): Use your own Gemini API key. Your browsing data is processed directly via the official Google AI API and is never stored on external servers.

[ HOW TO USE ]

1. Open the TabOrigami popup and set your Gemini API key in the integrated settings.
2. Optionally, select your preferred language (Japanese / English) and model name.
3. Select an "Origami Style" that fits your current needs.
4. Review the AI's suggestions in the preview list and adjust via drag-and-drop or checkboxes.
5. Click "Execute" to apply the grouping, or "Undo" to revert to the previous state.

[ DEVELOPER & POWER USER BENEFITS ]

- Supports the latest Gemini models (e.g., gemini-2.5-flash, gemini-3.5-flash). The model name is fully configurable.
- Advanced AI Context: The AI considers page content alongside titles and URLs for superior categorization accuracy.
- Built with Manifest V3 for modern security and performance standards.
- Domain-based grouping logic keeps related resources consolidated.
- Full UI language support: Japanese and English.

[ PRIVACY FIRST ]

TabOrigami functions as a client-side tool that communicates directly with the Google AI API using your personal key. We do not track, collect, or sell your browsing history. No data is sent to any servers other than the official Google AI API endpoint.

### **名前**
TabOrigami

### **短い説明**
Gemini AIを活用して、散らかったブラウザのタブを文脈に基づいて賢く整理します。

### **詳しい説明**
TabOrigamiは、Gemini AIを活用して、混沌としたタブバーを整然としたワークスペースへと変えるブラウザ拡張機能です。開いているページのタイトル、URL、および内容をAIが解析し、あなたのワークフローに最適なスタイルで自動的に分類・グループ化します。

【 主な機能 】

・AIによる自動整理: 「おまかせ」「タスク集中」「仕事とプライベート」「断捨離」の4つのスタイルから選択し、コンテンツに基づいた最適なグループ化を実現します。
・プレビューと微調整: AIの提案を適用前に確認できます。ドラッグ＆ドロップによる直感的な並び替えやグループ間の移動、チェックボックスによる個別のタブ選択が可能です。
・スマートクリーンアップ: 不要な「新しいタブ」を自動的に閉じる一方で、同一ドメインのタブはまとめて整理します。
・安心のUndo機能: 整理後に元の状態へ戻したい場合も、ワンクリックで即座に復元できます。
・固定タブの除外: ピン留めしたタブを整理の対象外にするオプションで、固定ワークフローのタブを保護できます。
・リアルタイムの進捗表示: AIによるタブ解析の進捗状況が可視化され、整理プロセスを明確に把握できます。
・プライバシー重視 (BYOK): ユーザー自身のGemini APIキーを使用する「Bring Your Own Key」方式を採用。データはGoogleの公式APIを通じて直接処理され、外部サーバーに保存されることはありません。

【 使用方法 】

1. TabOrigamiのポップアップを開き、設定画面からご自身のGemini APIキーを登録します。
2. 必要に応じて、表示言語（日本語 / English）と使用するモデル名を設定します。
3. 現在の作業状況に合った「折り方（整理スタイル）」を選択します。
4. プレビューリストでAIの提案内容を確認し、ドラッグ＆ドロップやチェックボックスで調整します。
5. 「実行する」ボタンでタブを整理します。必要に応じて「元に戻す」も利用できます。

【 開発者・パワーユーザー向けのメリット 】

・最新のGeminiモデルに対応（例: gemini-2.5-flash, gemini-3.5-flash）。モデル名は自由に設定可能です。
・高度なAIコンテキスト: ページのタイトルやURLだけでなく、ページの内容も考慮したプロンプト設計により、より高精度な分類を実現します。
・Manifest V3準拠による高いセキュリティとパフォーマンス。
・ドメインベースのグループ化ロジックで、関連リソースを効率的に集約。
・UI言語の完全切り替え対応（日本語 / English）。

【 プライバシーについて 】

TabOrigamiはクライアントサイドで動作し、個人のAPIキーを使用してGoogleの公式AI APIと直接通信します。閲覧履歴を追跡・収集・販売することはなく、Google AIの公式エンドポイント以外のサーバーにデータが送信されることもありません。
