# TabOrigami プライバシーポリシー

TabOrigami（以下、「本拡張機能」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーでは、本拡張機能が収集する情報、その利用目的、および情報の取り扱いについて説明します。

## 1. 収集する情報とその利用目的

本拡張機能は、以下の情報を取得・利用します。

- **ブラウザのタブ情報（URLおよびタイトル）**
  - **利用目的:** ユーザーの要求に基づき、タブをコンテキストに合わせてグループ化・整理（折りたたみ）するために利用します。
  - **取り扱い:** これらの情報は、整理ロジックを実行するために Google Gemini API に送信されます。
- **Gemini APIキー**
  - **利用目的:** AIモデル（Gemini）と通信するために利用します。
  - **取り扱い:** ユーザーが入力したAPIキーは、ブラウザのローカルストレージ（`chrome.storage.local`）にのみ安全に保存されます。

## 2. 情報の保存と共有

- **開発者による収集の否定:** 本拡張機能の開発者が、ユーザーのタブ情報（閲覧履歴を含む）、APIキー、またはその他の個人データを収集、保存、閲覧、または販売することは一切ありません。
- **第三者への提供:** 本拡張機能は、タブの整理機能を果たす目的でのみ、タブ情報を Google Gemini API に直接送信します。送信されたデータは Google のプライバシーポリシーに基づいて処理されます。その他の第三者へユーザーのデータを提供することはありません。

## 3. データの保護とセキュリティ

- APIキーおよびユーザー設定は、Google Chrome が提供するローカルストレージAPIを使用して、ユーザーのデバイス上にローカルでのみ保存されます。
- 外部サービス（Gemini API）への通信は、暗号化された安全な接続（HTTPS）を通じて行われます。

## 4. 同意

ユーザーは、本拡張機能を利用することにより、本プライバシーポリシーに記載されている情報の収集および利用に同意したものとみなされます。

## 5. プライバシーポリシーの変更

開発者は、必要に応じて本プライバシーポリシーを更新することがあります。重要な変更がある場合は、本拡張機能のアップデートノートやストアページを通じてお知らせします。

本プライバシーポリシーに関するご質問やお問い合わせは、本拡張機能のサポートページまたはストアの連絡先よりお願いいたします。
