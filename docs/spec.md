## 🕊️ TabOrigami - 基本仕様書

### 1. コアコンセプト

バラバラに広がったブラウザのタブ（紙）を、ローカルAI（Gemini Nano）が文脈を読み取り、ユーザーが選んだスタイルに合わせて綺麗に整理（折りたたみ）してくれる拡張機能。

### 2. トリガーとフロー

完全な手動実行型。勝手にタブは動かさない。

1.  **選択:** ポップアップから整理スタイル（折り方）を選ぶ。
2.  **プレビュー:** AIが分類結果を提示。ユーザーがチェックボックスで微調整。
3.  **実行:** 同意の上で整理を実行。
4.  **Undo:** 整理完了後、元の状態に戻せるボタンを表示。

### 3. 用意する「折り方」（AIへのプロンプト指示）

1.  **✨ おまかせ折り (Auto Context):** タブの文脈から最適なグループ名をAIが自動生成して分類。
2.  **💻 タスク集中折り (Task Focus):** 「現在のメイン作業」と「後で読む／ノイズ」に分離。
3.  **🏕️ オン・オフ折り (Work/Life Balance):** 仕事・開発関連と、趣味・プライベート関連に二分。
4.  **🧹 断捨離折り (Triage):** 保存すべき重要タブと、閉じてよさそうなタブを判別。

### 4. セーフティネット（Undo機能）

- **状態の記録:** 「実行」ボタンが押された瞬間のタブ一覧（URL, ID, 並び順など）を `chrome.storage.local` に一時保存（型紙として記録）する。
- **復元（ひらく）:** 整理直後の画面に「元に戻す（Undo）」ボタンを表示。押されたら、現在のタブを一旦クリア（または元のタブだけを復元）し、保存しておいた並び順通りにタブを展開し直す。
