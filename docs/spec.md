## TabOrigami - 基本仕様書

### 1. コアコンセプト

バラバラに広がったブラウザのタブ（紙）を、Gemini AI（クラウド版およびGemini Nano）が文脈（タイトル、URL、ページの内容など）を読み取り、ユーザーが選んだスタイルに合わせて綺麗に整理（折りたたみ）してくれる拡張機能。

### 2. トリガーとフロー

完全な手動実行型。勝手にタブは動かさない。

1.  **設定:** ポップアップ内に統合された設定画面でGemini APIキーを設定（画面遷移なし）。
2.  **選択:** ポップアップから整理スタイルを選ぶ。
3.  **処理中:** 各タブのコンテンツ（`og:description`、本文など）を読み取り、進捗をプログレスバーで可視化しながらAIが分類。
4.  **プレビュー:** AIが分類結果を提示。各タブのファビコンが表示され、ユーザーがチェックボックスで微調整。
    - 「断捨離」グループの横にはゴミ箱アイコンが表示され、閉じられるタブを視覚的に強調。
    - 「断捨離」スタイル時は、「保存すべきタブ」のチェックボックスは非表示となり、誤操作を防ぐ。対象タブがない場合は専用メッセージが表示され、実行ボタンは非表示になる。
    - 各タブ行の左端にドラッグハンドルが表示され、ドラッグ＆ドロップでグループ内の並び替えや別グループへの移動が可能。この並び替えは、実行後のブラウザ上のタブ順にも正確に反映される。
5.  **実行:** 同意の上で整理を実行。処理中（executing）でもキャンセルが可能。
6.  **Undo:** 整理完了後、元の状態に戻せるボタンを表示。

### 3. 用意するスタイル（AIへのプロンプト指示）

1.  **✨ おまかせ (Auto):** タブの文脈（タイトルやURLだけでなく、ページの内容も考慮）から最適なグループ名をAIが自動生成して分類。
2.  **💻 タスク集中 (Task Focus):** 「現在のメイン作業」と「後で読む／ノイズ」に分離。
3.  **🏕️ 仕事とプライベート (Work-Life):** 仕事・開発関連と、趣味・プライベート関連に二分。
4.  **🗑️ 断捨離 (Triage):** 保存すべき重要タブと、閉じてよさそうなタブを判別。不要なタブ（断捨離グループ）は自動的に閉じられる。

### 4. 設定項目 (Options)

- **ポップアップ統合:** 設定画面はポップアップ内に統合され、横幅500pxのUIで操作可能。ヘッダーはスクロール時も上部に固定。
- **APIキー:** Google AI Studioで発行したGemini APIキーを使用。
- **モデル選択:** 使用するGeminiモデルを指定（デフォルト: `gemini-3.1-flash-lite`）。
- **固定タブの除外:** 「固定されたタブを整理の対象外にする」設定が可能。

### 5. セーフティネット（Undo機能）

- **状態の記録:** 「実行」ボタンが押された瞬間のタブ一覧（URL, ID, 並び順）および、各タブが属していた「タブグループ情報（色やタイトル）」を `chrome.storage.local` に一時保存（型紙として記録）する。
- **復元（ひらく）:** 整理直後の画面に「元に戻す（Undo）」ボタンを表示。押されたら、単にタブを戻すだけでなく、元々属していたタブグループを再構築した上で、保存しておいた並び順通りにタブを展開し直す。
