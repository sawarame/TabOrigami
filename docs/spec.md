## TabOrigami - 基本仕様書

### 1. コアコンセプト

バラバラに広がったブラウザのタブ（紙）を、Gemini AI（クラウド版およびGemini Nano）が文脈を読み取り、ユーザーが選んだスタイルに合わせて綺麗に整理（折りたたみ）してくれる拡張機能。

### 2. トリガーとフロー

完全な手動実行型。勝手にタブは動かさない。

1.  **設定:** オプション画面でGemini APIキーを設定。
2.  **選択:** ポップアップから整理スタイルを選ぶ。
3.  **プレビュー:** AIが分類結果を提示。ユーザーがチェックボックスで微調整。
4.  **実行:** 同意の上で整理を実行。
5.  **Undo:** 整理完了後、元の状態に戻せるボタンを表示。

### 3. 用意するスタイル（AIへのプロンプト指示）

1.  **✨ おまかせ (Auto):** タブの文脈から最適なグループ名をAIが自動生成して分類。
2.  **💻 タスク集中 (Task Focus):** 「現在のメイン作業」と「後で読む／ノイズ」に分離。
3.  **🏕️ オン・オフ (Work/Life):** 仕事・開発関連と、趣味・プライベート関連に二分。
4.  **🧹 断捨離 (Triage):** 保存すべき重要タブと、閉じてよさそうなタブを判別。

### 4. 設定項目 (Options)

- **APIキー:** Google AI Studioで発行したGemini APIキーを使用。
- **モデル選択:** 使用するGeminiモデルを指定（デフォルト: `gemini-3.1-flash-lite`）。
- **固定タブの除外:** 「固定されたタブを整理の対象外にする」設定が可能。

### 5. セーフティネット（Undo機能）

- **状態の記録:** 「実行」ボタンが押された瞬間のタブ一覧（URL, ID, 並び順など）を `chrome.storage.local` に一時保存（型紙として記録）する。
- **復元（ひらく）:** 整理直後の画面に「元に戻す（Undo）」ボタンを表示。押されたら、現在のタブを一旦クリア（または元のタブだけを復元）し、保存しておいた並び順通りにタブを展開し直す。
